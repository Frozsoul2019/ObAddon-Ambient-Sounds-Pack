ObAddon "Prefab Sounds" Addon
By: Frozsoul
=============================

This addon adds ambient sounds to various ObAddon prefabs.

To use this addon:

	1. Place the addon PK3 file in your Oblige "addons" folder (where your 
		ObAddon PK3 file is located)
	2. Open the OBLIGE program and select this addon (Menu -> Addon List)
	3. Check the new "Prefab Sounds" option within the middle column.
	4. Build your Oblige map as usual
	
Note: This PK3 addon is meant only to be used with Oblige.  It should *not* be 
	included with your Doom launcher (e.g. GZDoom).

For more information about ObAddon and all available addons, join the OBLIGE 
	discord!
	A link to the Discord and more information about ObAddon is availble at:
	https://github.com/caligari87/ObAddon
